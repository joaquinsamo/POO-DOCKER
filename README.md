# POO-DOCKER
# Proyecto Aeroplano - POO + Docker

## 📌 Descripción
Aplicación Full Stack dockerizada que implementa un CRUD de Aeroplanos utilizando Node.js, Express y TypeScript en el backend y HTML + JS en el frontend.

---

## 🧱 Arquitectura

- Backend: Node + Express + TypeScript
- Frontend: HTML + JavaScript
- Contenedores: Docker + Docker Compose

---

## 🧩 Modelo POO

- Composición:
  - Cubierta
  - Tren de Aterrizaje

- Agregación:
  - Hélice
  - Alas
